SWR Pattern (stale-while-revalidate)

1. getElementById to get  DOM Elements into JS Variables

2. fetch requests, wrapped in easy to use functions

2.1 GET request, that trigger a render 
    
    function refresh() {
        fetch("...")
        .then((response) => response.json())
        .then((jsonData) => {
            state.xyz = jsonData;
            render();
        });
    }
